# Kiwoom Pro Algo-Trader v3.1

키움증권 OpenAPI+를 활용한 전문가용 자동매매 프로그램입니다. 변동성 돌파 전략을 기반으로 다양한 보조 지표(RSI, MACD, Bollinger Bands, ATR, DMI/ADX)를 필터로 사용하여 승률을 극대화합니다.

## 🚀 주요 기능 (v3.1)

### 1. 강력한 매매 전략
*   **변동성 돌파 전략**: 래리 윌리엄스의 검증된 단기 트레이딩 전략 적용
*   **트레일링 스톱**: 수익 보존을 위한 고점 추적 매도 기능
*   **다중 지표 필터링**:
    *   **RSI**: 과매수 구간 진입 방지
    *   **MACD**: 추세 확인 및 골든크로스 필터
    *   **Bollinger Bands**: 밴드 하단 돌파 시 매수 보류
    *   **ATR**: 변동성 기반 동적 손절 라인 설정
    *   **DMI/ADX**: 강력한 추세 장에서만 진입하도록 필터링
*   **거래량 분석**: 전일/평균 거래량 대비 급증 종목 포착

### 2. 리스크 관리 시스템
*   **일일 손실 한도**: 설정된 손실률 도달 시 당일 매매 자동 종료
*   **종목당 투자금**: 예수금 비율에 따른 자동 분산 투자
*   **최대 보유 종목 수**: 과도한 포트폴리오 확장 방지

### 3. 사용자 편의성
*   **모던 다크 테마**: 눈이 편안한 전문가용 UI 디자인
*   **실시간 대시보드**: KOSPI/KOSDAQ 지수, 실현 손익, 보유 종목 현황 실시간 모니터링
*   **텔레그램 알림**: 매수/매도 체결 및 프로그램 상태 실시간 모바일 알림 (v3.1 신규)
*   **프리셋 관리**: 공격적/표준/보수적 등 성향에 따른 전략 설정 저장 및 불러오기

## 📦 설치 및 실행 방법

### 요구 사항
*   Windows 10 또는 11 (32bit/64bit)
*   Python 3.8 이상 (32bit 권장 - 키움증권 OpenAPI가 32bit 기반)
*   키움증권 OpenAPI+ 설치 및 로그인

### 설치
1.  이 저장소를 클론하거나 다운로드합니다.
2.  필요한 라이브러리를 설치합니다:
    ```bash
    pip install pyqt5 requests
    ```

### 실행
```bash
python "키움증권 자동매매.py"
```

## ⚙️ 텔레그램 알림 설정 (신규)
1.  텔레그램에서 `@BotFather`를 검색하여 새 봇을 생성하고 **토큰(Token)** 을 발급받습니다.
2.  `@userinfobot` 등을 통해 본인의 **Chat ID**를 확인합니다.
3.  프로그램 실행 후 메뉴바의 `파일 > 시스템 설정`으로 이동합니다.
4.  `텔레그램 알림 사용`을 체크하고 토큰과 Chat ID를 입력합니다.
5.  저장하면 이후 매매 알림이 텔레그램으로 전송됩니다.

## 📁 파일 구조
*   `키움증권 자동매매.py`: 프로그램 메인 진입점 및 GUI
*   `config.py`: 프로그램 설정 및 상수 관리
*   `strategy_manager.py`: 매매 전략 및 지표 계산 로직 분리
*   `KiwoomProTrader.spec`: PyInstaller 빌드 설정 파일

*   모의투자를 통해 충분히 전략을 검증한 후 실전 매매에 사용하시기 바랍니다.
